# Services

Phase 1 services are package-map placeholders only.

## API

`services/api` is the future Java 25 + Spring Boot 4 modular monolith.

For Phase 1 final, Java source contains only package directories and `package-info.java` files. No controllers, entities, repositories, services, or runtime application class should be added until Phase 2.

## RAG

`services/rag` is the future Python local RAG/OCR service.

For Phase 1 final, it is a placeholder. Real Python service code starts in a later RAG phase.
