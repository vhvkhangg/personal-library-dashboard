# ADR 0005 — RAG Workspace UI Final

## Status

Accepted

## Decisions

- Left rail contains Settings, Chats, and Documents.
- Settings opens dialog.
- Documents owns upload progress and source selection.
- Chat header contains Model, Mode, and Search this chat.
- Citation numbers in answers are clickable.
- Right inspector contains Citation and Benchmark.
- Benchmark uses four threshold blocks.

## Deferred

Real document ingestion, OCR, embedding, retrieval, reranking, and generation are deferred.
