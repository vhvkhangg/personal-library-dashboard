# ADR 0001 — Foundation Decisions

## Status

Accepted

## Decisions

- Private single-user dashboard.
- Local self-hosting on Windows 11 with Docker Desktop.
- Modular monolith, not microservices.
- Next.js 16 frontend.
- Java 25 + Spring Boot 4 API.
- Python local RAG/OCR runtime.
- PostgreSQL + pgvector.
- Obsidian vault stays outside repo.
- Large media binaries stay outside PostgreSQL.
- Phase 2 begins with Auth.
