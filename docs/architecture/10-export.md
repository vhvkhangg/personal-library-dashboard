# Export

## Supported future export formats

- JSON
- CSV
- Excel
- WPS-compatible Excel output

## Export modes

- visible rows,
- current filtered list,
- favorites,
- full module,
- full backup later.

## Rules

- Do not include large binary media files by default.
- Export metadata and paths/references unless explicitly requested otherwise.
- Module exports may use multiple sheets.

## Phase 1

Export is UI-only.
